# GitHub User Activity CLI

Muestra la actividad pública reciente de cualquier usuario de GitHub desde la terminal.

Proyecto basado en [roadmap.sh](https://roadmap.sh/projects/github-user-activity).

## Instalación

```bash
git clone https://github.com/<tu-usuario>/github-user-activity.git
cd github-user-activity
```

## Uso

```bash
node github-activity.js <username>
```

**Ejemplo:**

```bash
node github-activity.js diegonais
```

## Requisitos

- Node.js v18 o superior (usa `fetch` nativo)
- Sin dependencias externas
